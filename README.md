# AI Content Generator

AI-powered content generation platform built using Flask, MySQL, BeautifulSoup, and Google Gemini.

## Features

* User Registration & Login
* Website Content Scraping
* AI Content Generation using Gemini
* MySQL Database Storage
* Article History Tracking
* PDF Export
* Modern Glassmorphism UI

## Technologies Used

* Python
* Flask
* MySQL
* BeautifulSoup
* Google Gemini API
* Bootstrap 5

## Installation

```bash
pip install -r requirements.txt
python app.py
```
